打开弹窗（模态），弹窗内容支持自定义设计。

### 动作说明

配置参数说明如下：

| 参数	| 说明	|
|  ----  | ----  |
| dialog	| 弹窗内容|

### 操作步骤
按以下步骤在页面设计器进行动作配置：
1. 打开动作配置弹窗后，选择弹窗消息 > 打开弹窗。
2. 选择类型，支持弹窗和抽屉两种类型。
3. 点击“去配置”，将弹出“配置弹窗内容”弹窗。
4. 配置弹窗内容后，确认提交。
